# FinTrack - Modern Personal Finance Management 💎

FinTrack is a high-performance, full-stack financial tracking application designed for individuals who want a clear, visual, and secure way to manage their money. Built with a focus on modern design principles (Glassmorphism) and real-time data visualization.

## 🚀 Key Features

- **Branded Dashboard**: Real-time stats for Balance, Income, and Expenses with intuitive period filtering (Monthly, Yearly, All Time).
- **Interactive Reports**: Deep-dive analytics including 6-month trend lines and categorical spending breakdowns.
- **Transaction Lifecycle**: Complete CRUD capabilities for transactions, including specialized 'Edit' and 'Delete' workflows.
- **Custom Categorization**: Users can define their own financial labels to tailor the tracking experience.
- **Premium UI/UX**: A responsive, glassmorphic interface that feels premium on both desktop and mobile devices.
- **Secure Architecture**: JWT-based authentication with bcrypt password hashing and Sequelize ORM for safe data persistence.

## 🛠️ Technical Implementation

### Frontend
- **React (Vite)**: Leveraging the performance and modularity of React for a snappy UI.
- **Tailwind CSS**: Custom-designed glassmorphism utility classes for a cohesive and modern look.
- **Chart.js & React-Chartjs-2**: High-quality, interactive data visualizations for trend analysis.
- **Lucide React**: Purposeful, modern iconography throughout the application.

### Backend
- **Node.js & Express**: A robust REST API architecture.
- **PostgreSQL & Sequelize**: Reliable relational data storage with an ORM for type-safe queries.
- **JSON Web Tokens (JWT)**: Secure user sessions and protected API routes.

## 📈 Impact
FinTrack simplifies financial awareness by turning raw transaction data into actionable insights through visual charts and organized summaries.
